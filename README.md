## 命名规范说明

每道题所在目录的命名格式如下：

**cf2066A-object-identification-1400/**

各部分含义如下：

- **cf**：平台缩写（Codeforces）
- **2066A**：题号及题目编号
- **object-identification**：题目英文标题（全小写，单词用短横线 `-` 连接）
- **1400**：该题的 Codeforces Rating（难度）

### ✅ 示例目录结构

    - `cf2066A-object-identification-1400/`
      - `README.md`：解题思路说明
